# Changelog

## [7.0.0-native] - 2026-07-17

### Added

- V7 原生微信小程序，包含今日、教练、打卡、数据、训练和饮食六个页面。
- A/B 训练轮换、逐组记录、RPE、休息计时和下次重量建议。
- 健康数据、打卡成就、本地规则教练和每周报告。
- 本地饮食估算确认流程，以及完整 JSON 备份与安全恢复。
- V5–V7 网页版本预览中心。

### Security

- 健康输入限制在合理数值范围。
- 备份恢复对训练、饮食、身体和进度字段执行嵌套白名单。
- 仓库不包含真实用户健康数据、照片、模型密钥或微信密钥。

### Known limitations

- 真实图像理解接口和云端同步尚未接入。
- 发布前仍需配置正式 AppID、隐私指引并完成微信真机验收。
